Gridea 添加 Valine 留言系统

2020-05-16 1 min read # Gridea

国内版 leancloud 要求实名认证（辣鸡），本次使用 leancloud 国际版。

主要步骤参考 Valine 文档 。

  1. 注册登录/创建应用
  2. 获取APP ID 和 APP Key
  3. 填入 HTML 片段
  4. 将修改后的 HTML 片段添加在 Gridea 模板页面(head.ejs/post.ejs等)
  5. 同步 Gridea

💊 根据这里 介绍，国际版 leancloud 需要手动在leancloud应用-存储中添加Comment这个class

👍 完成~
